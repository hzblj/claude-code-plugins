# Graphics & Advanced Rendering (Skia)

- Use React Native Skia for heavy graphical or canvas-based work.
- Prefer Skia for:
  - shaders
  - charts
  - complex drawing
  - GPU-driven effects
- Do not simulate complex graphics using nested Views.
- Consider GPU vs CPU cost for every visual effect.
